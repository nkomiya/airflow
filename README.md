# util

Utility modules for airflow

## Installation

It is assumed that this module is located in the directory `dependencies/`.

Directory structure might look like as follow.

```
dags/
├── dependencies/
│   ├── __init__.py
│   ├── util (this module)
│   │   └── ...
│   └── {{ other modules }}
└── {{ Python scripts define DAGs }}
```
